# MyTheresa

MyTheresa Frontend UI Challange.

- Monorepo ✅
- React w/a CRA ✅
- Sass ✅
- Redux ❌
- Context ✅
- SSR ❌

# Setup Instructions

- preferrable/recommended npm client (Yarn)
- Run `yarn install`
- Run `yarn dev`

# Available Script

- yarn dev - to start the app in development mod
- yarn build - to build for production
- yarn test - to test
